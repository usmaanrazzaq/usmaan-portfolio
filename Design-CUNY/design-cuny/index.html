Overview

Design @ CUNY is a website that allows students to explore different design programs at CUNY. The website is designed to be a resource for students to get an overall view of the design programs at different CUNY colleges. The website also provides direct links to the design programs admissions center and resources for students to get started with their design journey.

Process

For this project I used an excel sheet to collect information and data of each program. I then used the data to create the webite by placing each college information and program information within the right page.

Design

For the design process I focused on creating a simple and modern website that is easy to navigate and understand. The main focus was to create a website that put more focus on the programs and the resources available.
